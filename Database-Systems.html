Database Systems

Database Systems is my favorite book for learning SQL and the principles of Database design. SQL is the primary language used for talking to a database, and this book explains the language in detail. Where you are a novice or advanced SQL developer, you will learn a lot from this book. Topics include SQL,relational databases, indexes, and database design. This book is perfect for beginners and yet advanced enough for experienced database professionals. I used this book myself when I took my first database course.
